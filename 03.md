# A simple example -- singly linked list or a "cons list"

_You've probably implemented it few times already._

┌────────────┐     ┌────────────┐     ┌────────────┐
│  1  | tail │   ┌─│  2  | tail │   ┌─│  3  | null │
└────────│───┘   │ └────────│───┘   │ └────────────┘  
         └───────┘          └───────┘          
          
           a linked list of (1, 2, 3)
